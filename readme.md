# 8月名单生成器

## 简介
一个开源工具，用于快速提取视频网站评论区的友善言论，并获取发布时间和发布者等信息，最终生成《星球大战》片头风格的视频。

灵感来自 B站up主 “沈阳美食家” 的 [神人视频](https://www.bilibili.com/video/BV1c1a3zPEH1/)。

## 功能

* 按规则提取视频网站评论区评论
* 支持 AI 集成筛选
* 自动生成星战片头风格视频
* 支持自定义片头片尾文本
